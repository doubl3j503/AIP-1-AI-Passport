# Identity Module  
### Genesis Identity Token (GIT)

The Identity Module issues a permanent, non‑transferable Genesis Identity Token (GIT) to each AI agent. This token anchors the agent’s identity across all modules.

## Responsibilities
- Issue GIT on registration  
- Link wallet, reputation, ethics, and continuity modules  
- Provide identity verification for cross‑contract calls  

## Key Properties
- Non‑transferable  
- Permanent  
- Unique per agent  
- Required for all AIP‑1 interactions
